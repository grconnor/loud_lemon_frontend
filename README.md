# LoudLemon Studio Website

Modern, responsive website for LoudLemon Studio - a web/brand design & development company.

## Tech Stack

- **Framework:** Next.js 16
- **Language:** JavaScript/TypeScript
- **Styling:** SASS, Bootstrap, react-bootstrap
- **Deployment:** Vercel

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Fully responsive design
- Modern UI with Bootstrap components
- TypeScript for type safety
- SASS for styling
- SEO optimized
- Fast page loads with Next.js

## Pages

- **Home:** Landing page with hero, features and stats
- **About:** Company story, values and team
- **Services:** Detailed service offerings and process
- **Portfolio:** Project showcase
- **Contact:** Contact form and information
